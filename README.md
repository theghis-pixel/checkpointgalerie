# Galerie Produits (HTML / CSS / JS)

## Description
Projet de galerie de produits interactive réalisé sans framework.  
Il inclut une bascule de thème, un filtrage par catégories, des animations et une lightbox pour afficher les images en grand.

---

## Fonctionnalités

- Galerie de produits en grille responsive
- Animations d’apparition (fade-in)
- Effet de survol sur les cartes et images
- Bascule thème clair / sombre
- Sauvegarde du thème (localStorage)
- Filtrage des produits par catégorie
- Lightbox (clic pour agrandir une image)

---

## Technologies utilisées

- HTML5
- CSS3 (variables CSS, animations, grid)
- JavaScript (DOM, events, localStorage)

---

## Installation

Aucune installation requise.

1. Télécharger le fichier `index.html`
2. Ouvrir dans un navigateur
   ou
   utiliser un éditeur comme Replit / VS Code Live Server

---

## Utilisation

### Changer le thème
Cliquer sur le bouton "Mode sombre / Mode clair"

### Filtrer les produits
Utiliser le menu déroulant pour afficher une catégorie

### Agrandir une image
Cliquer sur une image pour ouvrir la lightbox
